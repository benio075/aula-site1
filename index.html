Linguagens e suas Utilidades

JavaScript

JavaScript foi criado em 1995 por Brendan Eich enquanto trabalhava na Netscape, inicialmente chamado de Mocha e depois renomeado para JavaScript. É uma linguagem interpretada, dinâmica e baseada em protótipos, projetada para rodar em navegadores web, permitindo manipulação dinâmica de conteúdo HTML e CSS. Além disso, possui tipagem fraca e funções de primeira classe, o que traz grande flexibilidade. Com o surgimento do Node.js, JavaScript passou a ser usado também no lado servidor, possibilitando o desenvolvimento completo de aplicações web, desde interfaces interativas até backends escaláveis.

console.log("Olá Mundo");

Python

Python foi criado no final da década de 1980 por Guido van Rossum e lançado em 1991, com o objetivo de ser uma linguagem fácil de ler e escrever. Suas principais características incluem sintaxe simples e clara, tipagem dinâmica e forte, além de um vasto ecossistema de bibliotecas. Python é uma linguagem multiparadigma que suporta programação procedural, orientada a objetos e funcional. É amplamente utilizada em ciência de dados, inteligência artificial, automação, desenvolvimento web e scripting, devido à sua versatilidade e facilidade de integração com outras tecnologias.

print("Olá Mundo")

C#

C# foi desenvolvido pela Microsoft no início dos anos 2000, liderado por Anders Hejlsberg, como parte da plataforma .NET. É uma linguagem de programação orientada a objetos, fortemente tipada e com suporte a programação assíncrona, LINQ e coleta automática de lixo. C# é muito usada para criar aplicações desktop no Windows, desenvolvimento de jogos com a engine Unity, e aplicações web através do framework ASP.NET. Seu design moderno e integração com o ecossistema Microsoft fazem dela uma linguagem robusta e confiável para software corporativo.

                     }

Java

Java foi criada pela Sun Microsystems em 1995, idealizada por James Gosling, com o foco em portabilidade e segurança, adotando o conceito “write once, run anywhere”. Java é uma linguagem orientada a objetos, fortemente tipada, com gerenciamento automático de memória via coleta de lixo. Seu ambiente de execução (JVM) permite que o código seja executado em qualquer plataforma que tenha a JVM instalada. Java é largamente usada em sistemas empresariais, aplicações Android, servidores web, e sistemas distribuídos, por sua estabilidade e escalabilidade.

C++

C++ foi desenvolvido por Bjarne Stroustrup no início dos anos 1980 como uma extensão do C, adicionando suporte à programação orientada a objetos. Combina características de baixo nível, permitindo manipulação direta de memória e hardware, com recursos de alto nível para abstração e organização do código. C++ é amplamente usado em sistemas operacionais, desenvolvimento de jogos, software de alta performance, sistemas embarcados e aplicações que exigem controle rigoroso sobre recursos computacionais.

Assembly

Assembly é uma linguagem de baixo nível que surgiu junto com os primeiros computadores digitais, representando instruções quase diretamente em código máquina específico para cada arquitetura de processador. Sua principal característica é o controle absoluto sobre o hardware, permitindo otimização máxima de desempenho e manipulação direta de registradores, memória e dispositivos. Assembly é essencial para desenvolvimento de firmware, drivers, sistemas embarcados e para entender profundamente o funcionamento interno dos computadores, embora sua complexidade e dificuldade limitam seu uso no desenvolvimento geral.

section .data
                     msg db "Olá Mundo", 10
                     len equ $ - msg
                 section .text
                     global _start
                 _start:
                     mov rax, 1        ; syscall: write
                     mov rdi, 1        ; stdout
                     mov rsi, msg
                     mov rdx, len
                     syscall
                     mov rax, 60       ; syscall: exit
                     xor rdi, rdi
                     syscall
